# Chinmay's Homebrew Tap

## Usage

### Tap and Install New Packages
```sh
brew tap chinmayarde/tap
brew install doxygen2docset
```
### Install Package Directly

```sh
brew install chinmayarde/tap/doxygen2docset
```

### Use in a `brew bundle` `Brewfile`

```sh
tap "chinmayarde/tap"
brew "doxygen2docset"
```
